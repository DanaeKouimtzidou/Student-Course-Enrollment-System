# Student Course Enrollment System (Java Swing)

## Overview

This project is a simple Java Swing application that allows users to enroll students in predefined courses. It demonstrates Object-Oriented Programming (OOP) concepts together with Java Swing for building graphical user interfaces.

The application lets users enter a student's name, ID, and course, enroll the student, and display course information.

## Features

- Graphical User Interface (Java Swing)
- Student enrollment
- Predefined course list
- Display course details
- Input validation
- Object-Oriented design

## Classes

- **Student**
  - Stores student information (name and ID).

- **Course**
  - Stores course information and enrolled students.

- **EnrollStudentFrame**
  - Implements the graphical interface.
  - Handles user interactions.

- **Main**
  - Starts the application.

## How to Run

1. Clone the repository.
2. Open the project in Eclipse.
3. Run `Main.java`.
4. Enter:
   - Student Name
   - Student ID
   - Course Name
5. Click **Enroll Student**.
6. Click **Print Course Details** to display enrolled students.
